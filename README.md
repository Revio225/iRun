# 🏃‍♂️ iRun - Aplikasi Pencatat Aktivitas Lari

Aplikasi Android sederhana berbasis **Kotlin** untuk mencatat progres latihan lari harian pengguna. Proyek ini dibuat untuk memenuhi tugas Pemrograman Mobile.

## 📱 Fitur Utama
1.  **Halaman Login :** Gerbang masuk aplikasi dengan validasi sederhana.
2.  **Dashboard (Beranda) :** Menampilkan ringkasan statistik (Total Jarak & Total Aktivitas) serta sapaan pengguna.
3.  **Input Data Lari :** Form untuk memasukkan Tanggal, Jarak (km), dan Waktu tempuh.
4.  **Navigasi :** Alur perpindahan antar halaman menggunakan `Intent`.
5.  **Profil :** Tampilan biodata pengguna.
